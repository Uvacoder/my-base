# Memulai

## React

### `create-react-app`
```bash
npx create-react-app <nama_aplikasi>
## or
yarn create react-app <nama_aplikasi>

# typescript
npx create-react-app <nama_aplikasi> --template typescript
## or
yarn create react-app <nama_aplikasi> --template typescript
```
referensi: 
- [reactjs.org](https://reactjs.org/docs/create-a-new-react-app.html)
- [create-react-app docs](https://create-react-app.dev/)
- [adding typescript](https://create-react-app.dev/docs/adding-typescript/)

### Some react template to quickly initialize projects and start developing right away:

#### nextchakra-starter

nextchakra-starter adalah sebuah template untuk memulai projek aplikasi [**Next.js**](https://nextjs.org/) yang sudah PWA-ready (dimatikan secara default) serta sudah dipasangkan dengan [**Chakra-UI**](https://chakra-ui.com/) dan [**TypeScript**](https://www.typescriptlang.org/).

**Deploy Langsung**
[![Deploy dengan Vercel](https://vercel.com/button)](https://vercel.com/import/git?s=https://github.com/sozonome/nextchakra-starter) [![Deploy ke Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/sozonome/nextchakra-starter)

**menggunakan `create-next-app`**:
```bash
npx create-next-app <nama_aplikasi> --example https://github.com/sozonome/nextchakra-starter
# or 
yarn create next-app <nama_aplikasi> --example https://github.com/sozonome/nextchakra-starter
```

or [**create a new repository**](https://github.com/sozonome/nextchakra-starter/generate) from nextchakra-starter | [repository](https://github.com/sozonome/nextchakra-starter)
