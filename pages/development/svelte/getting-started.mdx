# Getting Started

## Svelte
```bash
npx degit sveltejs/template <project_name>
cd <project_name>

npm i
npm run dev
```

reference: [svelte.dev](https://svelte.dev/blog/the-easiest-way-to-get-started#2_Use_degit)

### `create-snowpack-app`
```bash
npx create-snowpack-app <app_name> --template @snowpack/app-template-svelte-typescript [--use-yarn | --use-pnpm | --no-install]
npx create-snowpack-app <app_name> --template @snowpack/app-template-svelte
```

references:
- [https://github.com/snowpackjs/snowpack/tree/main/create-snowpack-app/cli](https://github.com/snowpackjs/snowpack/tree/main/create-snowpack-app/cli)
